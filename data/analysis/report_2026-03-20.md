## 📈 技术趋势与演进

### 2. 技术趋势与演进

*   **主流方法与架构**：
    *   **优化方法**：一阶随机梯度下降（SGD）及其变体仍是主流，但对其**改进焦点**从单纯加速收敛（如自适应学习率）转向**改善训练动态**（如谱裁剪）和**增强泛化**（如新的正则化技术）。
    *   **学习范式**：**分割学习（Split Learning）** 与**联邦学习**的融合成为解决隐私与效率矛盾的主流架构思路（如HO-SFL）。模型在客户端和服务器间的切割策略、聚合算法是核心。
    *   **理论工具**：**凸优化理论**、**动力系统**、**随机过程**仍然是分析算法收敛性和泛化性的基石。

*   **正在兴起的新技术**：
    *   **零阶优化（Zeroth-Order Optimization, ZOO）的复兴与实用化**：ZOO不再是理论玩具，正被系统性地应用于**测试时适配（TTA）**（论文6：ZOTTA）、**黑盒对抗攻击/防御**、**联邦学习**等前沿场景。其与**贝叶斯优化**、**进化策略**的交叉值得关注。
    *   **基于理论指导的稳定化技术**：如论文2中源自内点法的**对数障碍（Log-Barrier）方法**用于RL探索，论文5中控制Hessian谱半径的**谱裁剪**。它们比启发式方法（如梯度裁剪）更具可解释性和理论保证。
    *   **拓扑与几何视角的网络分析**：将神经网络各层视为“层与层之间的映射”，用代数拓扑研究其**不变性**和**表达能力**（论文3），这是一个非常新颖且强大的视角。

*   **技术演进路径**：
    1.  **从集中到分散**： 单机训练 -> 分布式训练 -> 联邦/去中心化学习 -> **混合阶、无反向传播的高效协作学习**。
    2.  **从启发式到原理性**： 经验性的调参（如学习率调度）-> 基于优化理论的设计（如自适应优化器）-> **基于更深数学理论（动力系统、控制论、拓扑）的训练过程控制**。
    3.  **从有梯度到无梯度**： 依赖精确梯度 -> 处理次梯度、近似梯度 -> **在广泛场景中系统应用零阶优化**，作为梯度不可得时的通用替代方案。
    4.  **从性能驱动到多维均衡**： 追求SOTA精度 -> 同时兼顾**隐私、通信开销、计算开销、内存占用、泛化能力**的均衡设计。
